# Netflix IMDb Insights Dashboard
The Netflix IMDb Insights Dashboard is a visually rich Power BI report built to help users analyze trends in movie and TV show performance over the last 50 years. It highlights top-rated titles, decade-wise content distribution, rating patterns, voting behavior, and certification-based comparisons.
This dashboard is intended for content strategists, media analysts, entertainment researchers, and data-driven decision-makers seeking deeper insights into long-term patterns in the global film industry.,

# Tech Stack
The dashboard was built using the following tools and technologies:
Power BI Desktop
Main platform used to design visualizations, dashboards, and storytelling elements.
Power Query
Used for data cleaning, transforming raw IMDb data, and preparing structured tables.
DAX (Data Analysis Expressions)
Created calculated measures to analyze IMDb ratings, vote distributions, runtimes, and category-based comparisons.
Data Modeling
Established relationships among tables (movies, shows, ratings, certifications, time-based data) to enable accurate cross-filtering and dynamic exploration.
File Formats
.pbix for development
.png for dashboard previews and project documentation

# Data Source
Source: IMDb public datasets & curated data files for movies and TV shows (spanning 50 years).
Dataset Includes:
*Movie and TV show titles
*IMDb ratings & vote counts
*Release year (decade-wise segmentation)
*Age certifications (G, PG, PG-13, R, etc.)
*Runtime duration
*Genres
The dataset is structured to support exploratory analysis, trend identification, and viewer engagement insights.
